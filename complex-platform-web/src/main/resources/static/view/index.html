后台管理框架

此后台管理系统源于若依后台管理系统，在此进行二次开发，更适合自己使用。 在项目中添加一些第三方插件与功能，方便大家使用，如果觉得有用，希望可以帮忙传播。

当前版本：v1.0

技术选型：

  1. 核心框架：Spring Boot。
  2. 安全框架：Apache Shiro。
  3. 搜索引擎：ElasticSearch。
  4. 模板引擎：Thymeleaf。
  5. 持久层框架：MyBatis。
  6. 定时任务:Quartz。
  7. 数据库连接池：Druid。
  8. 工具类：Fastjson。
  9. 更多……
